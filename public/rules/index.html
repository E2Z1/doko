Spielverlauf

Sonderpunkte

Fuchs

Ein Karo-Ass wird als ein Fuchs bezeichnet. Wird dieser von der gegenerischen Partei gestochen, erhält diese je einen Sonderpunkt (Fuchs gefangen).

Klabautermann

Wird ein Pik-König von einer Pik-Dame gestochen, die den Stich gewinnt, erhält die Partei, die die Pik-Dame gelegt hat, einen Sonderpunkt. Wird der Pik-König nach der Pik-Dame gelegt, so zählt das nicht mehr als ein Klabautermann. Die Partei erhält auch einen Punkt, wenn der Pik-König vom Mitspieler stammt.

Karlchen

Ein Kreuz-Bube wird als ein Karlchen bezeichnet. Ist dieser die höchste Karte im letzten Stich, erhält die Partei, die das Karlchen gelegt hat einen Sonderpunkt. Erhält die andere Partei jedoch den Stich, in welchem das Karlchen ist, erhält die andere Partei einen Sonderpunkt.

Doppelkopf

Enthält ein Stich 40 oder mehr Punkte wird dieser als Doppelkopf bezeichnet. Die Partei, die diesen Stich gewinnt, erhält einen Sonderpunkt.

Koppeldopf (Neunerstich)

Enthält ein Stich 0 Punkte wird dieser als Koppeldopf (oder Neunerstich) bezeichnet. Die Partei, die diesen Stich gewinnt, erhält einen Sonderpunkt.
